## Css变量&Css动画

声明Css变量 用  --  声明

```css
body {
  --foo: #7F583F;
  --bar: #F7EFD2;
}
```

有点像之前小程序写scss的时候定义全局的变量。

写css的时候 使用变量用 var（变量）

`var()`函数还可以使用第二个参数，表示变量的默认值。如果该变量不存在，就会使用这个默认值。

> ```css
> color: var(--foo, #7F583F);
> ```
